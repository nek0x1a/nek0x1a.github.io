# 侧边栏

Dcusaurus 的侧边栏配置文件是 `sidebars.ts`。

通过一个 `[id: SidebarsConfig]` 数组将配置文件传入。

一般情况下，自动根据文件夹结构生成即可：

```typescript
[{ type: "autogenerated", dirName: "." }];
```

也可以手动创建侧边栏：

```typescript showLineNumbers
[
  "intro",
  "hello",
  {
    type: "category",
    label: "Tutorial",
    items: ["tutorial-basics/create-a-document"],
  },
];
```
